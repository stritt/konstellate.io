Introducing Konstellate

Konstellate is a UI to create edit and manage Kubernetes resources and their relationships. You can easily create complex YAML and export them as Helm charts or Kustomize templates.

Try it now

Create

Construct complex Kubernetes resources without ever touching YAML.

Connect

Drag a line between resources and Konstellate will show you all the possible connections.

Customize

Automatically generate overlay variations based on your different workspaces.

Compose

Konstellate allows you to import and export from YAML, Kustomize, or Helm.
